Procedure to Create and View a Website on The Production Server

  1. Open Terminal
  2. Clone your repository.
  3. Make sure you are in your repository
  4. Use the command python -m SimpleHTTPServer to start localhost:8000
  5. Use gedit to open your file
  6. Edit the html file first
  7. Make sure you start with !doctype html
  8. Make sure you include header,body,nav,and footer.
  9. Put in a title with h1
  10. Make sure you close off all your commands with a /
  11. p make paragraphs
  12. h1, h2, etc.. make gradual headers that go down in size if the number is bigger.
  13. Style your webpage with a css file.
  14. once you're done use sftp username@cmis-hsteacher
  15. use put {filename} to upload a file.
  16. add, commit, push.
